# MediCore

MediCore is a full-stack MVP Hospital Management Platform with a React + Vite frontend and a FastAPI backend. The app uses mock data only (no database, no authentication).

## Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

## Frontend

```bash
cd frontend
npm install
npm run dev
# runs on http://localhost:5173
```
